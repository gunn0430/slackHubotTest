# hubot-esbot

A hubot script that does the things

See [`src/esbot.coffee`](src/esbot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-esbot --save`

Then add **hubot-esbot** to your `external-scripts.json`:

```json
[
  "hubot-esbot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
